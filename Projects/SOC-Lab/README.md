# SOC Lab & Microsoft Security Operations

## Overview

This project documents the design and implementation of a hands-on Security Operations Center (SOC) lab built using Microsoft security technologies. The lab was created to develop practical experience in security monitoring, endpoint protection, threat detection, incident investigation, and incident response within a Microsoft cloud environment.

## Objectives

- Build a functional SOC lab environment
- Configure Microsoft security services
- Onboard and secure endpoints
- Monitor security events and alerts
- Investigate incidents and security findings
- Develop practical SOC analyst skills

## Technologies Used

- Microsoft Defender for Endpoint
- Microsoft Sentinel (SIEM)
- Microsoft Intune
- Microsoft Entra ID (Azure AD)
- Azure Log Analytics
- Microsoft 365 Defender

## Key Activities

- Microsoft 365 tenant configuration
- Defender service deployment
- Endpoint onboarding and management
- Security policy implementation
- Endpoint Detection and Response (EDR) configuration
- Web content filtering
- Identity and Access Management (IAM)
- Security monitoring and alert review
- Incident investigation and response

## Skills Demonstrated

- Security Operations (SOC)
- Threat Detection and Analysis
- Incident Response
- Endpoint Security
- Security Monitoring
- Log Analysis
- Microsoft Security Administration
- Identity and Access Management
- Security Architecture Fundamentals

## Key Outcomes

- Built and managed a Microsoft-based SOC environment
- Gained practical experience with security monitoring and investigations
- Developed hands-on skills using Microsoft Defender and Sentinel
- Improved understanding of modern security operations workflows
