## Overview

The TryHackMe SOC Level 1 Path is a practical, hands-on training program designed to develop the technical and analytical skills required in a Security Operations Centre (SOC) environment.

Through guided labs, realistic investigations, alert triage exercises, and threat analysis scenarios, I have gained practical exposure to the core responsibilities of a SOC Analyst, including security monitoring, phishing analysis, network traffic investigation, threat detection, incident handling, and security operations workflows.

Unlike traditional theory-based learning, this path emphasizes practical application through simulated real-world security events and analyst investigations.
